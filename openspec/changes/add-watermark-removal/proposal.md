# Change: 实现去水印功能（视频和图片）

## Why
去水印是本小程序的核心功能，用户需要能够去除视频和图片中的水印。需要使用开源框架实现这一功能，支持两种场景：
1. 短视频平台链接解析（抖音、快手、小红书等）- 获取无水印原视频
2. 本地图片/视频的水印去除 - 使用图像处理技术

## What Changes
- 实现 URL 解析去水印服务（解析短视频平台无水印源）
- 实现本地图片去水印功能（使用 Canvas / 服务端处理）
- 实现本地视频去水印功能（服务端处理）
- 新增去水印结果页面
- 新增去水印历史记录功能
- 集成开源去水印框架/API

## Impact
- Affected specs: `watermark-removal` (新增)
- Affected code:
  - `src/pages/` - 新增去水印结果页、历史记录页
  - `src/services/` - 新增去水印服务
  - `src/utils/` - 新增解析工具函数
  - `server/` - 可能需要服务端支持（可选）
